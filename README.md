# Pocket Utility – One-Page Vue (CDN) App

- **Summary:**
  - Mobile-first utility app for tackling common daily struggles with actionable mindset shifts and micro-tasks

---

- **Tech Stack:**
  - Vue.js (via CDN, no build tools)
  - HTML5
  - Tailwind CSS (via CDN)
  - JavaScript (ES6+)
  - Local Storage API

---

- **Key Features:**
  - Interactive, single-page mobile experience
  - Vue reactivity for dynamic UI updates
  - State management using Vue's data and methods
  - Component-like structure via Vue CDN (no SFCs)
  - Persistent user data with Local Storage
  - Animated, engaging UI elements
  - Responsive design (mobile-first)
  - Clean separation of logic and presentation

---

- **Architecture Overview:**
  - No build tools or bundlers; pure CDN imports
  - All logic and UI in a single HTML file
  - Vue instance manages state, actions, and UI reactivity
  - Tailwind CSS for rapid, utility-first styling
  - JavaScript modules and functions for reusable logic patterns
  - Clear separation of concerns: template, script, and style blocks

---

- **What I Learned / Skills Demonstrated:**
  - Vue.js reactivity and state management without build tools
  - DOM manipulation and event handling in a CDN context
  - Component thinking using Vue instance options
  - Performance awareness in single-file, CDN-based apps
  - Responsive/mobile-first UI engineering
  - Local Storage integration for persistence
  - Clean, maintainable code structure without frameworks
